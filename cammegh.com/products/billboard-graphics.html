Billboard Graphics

The Billboard Display System complements any gaming table by displaying results, statistics and promotional material in pin-sharp high definition. Game data can be obtained automatically from any Cammegh roulette wheel or read from a 3rd party wheel via the EyeBall system, or for card based games via the EyeCard system. For manual entry, the Billboard's supplied numeric keypad or the enhanced TD3 Keypad can be used to add and remove game results, as well as provide access to many other features.

Cammegh Billboard Displays have always raised the bar with respect to the features and quality a gaming table display should offer. Each Billboard offers a blank canvas and a wealth of configurable elements with which to fill it. Billboards can hold up to 100 separate screen layouts, so there is plenty of room for experimentation.

For Spread-Bet Roulette, Spread-Bet Blackjack, Spread-Bet Baccarat, Lucky Symbols Roulette and Double Shot, see our licences page.

Personalised Animations

High quality 'No More Bets' and 'Winning Number' animations can be included, each customised to match the table's chips, cloth and roulette wheel. If further personalisation is required, additional animations can be made to order.

Paytables

The Billbaord's dynamic paytable system uses colour to highlight table denominations, helping players to find the table that's right for them.
Each Billboard can store up to 100 separate paytables, each of which can display anything from simple min and max values to a full table of odds, house edges and game rules. Paytables can be locked in as presets or edited at the table using the on-screen user menu.
As with all Billboard software elements, paytables can be customised extensively; each one can be configured to change the colour of the on-screen graphics and (where applicable) the LED side strips, with millions of colours available.

Statistics

A wide variety of statistics are available for all games on the Billboard Display System, all of which can be customised to suit any application. Custom statistics can also be developed, subject to requirement.

Layout Options

Casino Logo
The casino's logo can be incorporated into the display to enhance brand identity.
Live Video
Billboards can receive multiple live video streams (or S-video/Composite video inputs) and display them anywhere on the screen at any size. Multicasts in MPEG2 and H.264 can be added to the on-screen user menu, to enable easy switching between channels at the table.
Chip Tree
Chip Tree makes a stylish and practical addition to the Billboard. Chip values can be entered using quick-access keys on the keypad provided, and both their design and colour can be styled to match any chips in play.
Pay Tables
Betting limits can be preloaded into the display, allowing the dealer to quickly change paytable when necessary.

Trends

Player and Banker trends can be highlighted as they occur, adding exitement at the table and drawing in new players. In the exampls above, the standard display graphics (left) change to show fiery effects (centre and right) when runs of Player and Banker results occur.

Card Display

The cards in play can be shown on-screen using Billboard's card display add-on, available for Billboards using an Angel Eye or Bee-Tek card shoe, or using the Cammegh EyeCard system. Cards are arranged either traditionally (as shown above) or in fans, and can appear either as they are drawn or when the reveal button is pressed. For Billboards using EyeCard, cards appear on-screen as they are revealed.

Blackjack

Blackjack results and statistics can be displayed live using the Cammegh EyeCard system or entered manually through the Billboard's numeric keypad. Result animations, statistics and a history stack can be incorporated for the dealer's hand.
